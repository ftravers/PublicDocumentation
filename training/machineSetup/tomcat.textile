h2. Extract

p(rightsidebox). !(img_rightsidebox)/images/mac.jpeg! !(img_rightsidebox)/images/linux.jpeg! For mac and linux users copy the _tar.gz_ to a folder such as: _~/bin/tomcat-6.0.24.

( '~' or tilde is a shortcut for your home directory.  My home directory is: @/home/fentontravers@, yours will be @/home/[yourUserId]@)

p(rightsidebox). !(img_rightsidebox)/images/windows.jpeg! For windows users select just accept the default install directory Apache Installer recommends.
